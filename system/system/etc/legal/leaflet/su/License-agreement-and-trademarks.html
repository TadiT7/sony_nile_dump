Perjangjian lisensi pamake ahir

Software nu dikirimkeun ku alat ieu jeung mediana eta teh boga Sony Mobile Communications Inc., jeung/atawa pausahaan apiliasina jeung pamasokna jeung nu gaduh lisensina.

Sony mere idin lisensi non-ekslusip kawatesan ka Anjeun jang make software bareng jeung alat nu dipasangan atawa nu dikirimkeun. Hak ngabogaan software ieu teu bisa dijual, dibikeun ka batur atawa disebutkeun salain eta.

Ulang make cara naon wae jang nempo sumber kode atawa komponen software, nyieun jeung mikeun software, atawa ngarobah software. Anjeun boga hak jang mikeun hak jeung kawajiban kana software ka pihak katilu, ngan kudu bareng jeung alatna nu aya software eta, pihak katilu satuju sacara tulisan yen manehna kaiket ku sarat lisensi ieu.

Lisensi ieu aya salila alat ieu bisa keneh dipake. Lisensi ieu bisa dicabut ku cara mikeun hak Anjeun kana alat ieu ka pihak katilu sacara tulisan.

Gagal jang nurut kana sarat jeung katangtuan ieu bakal nyabut lisensi na harita oge.

Sony jeung pamasok pihak katiluna jeung nu gaduh lisensina ngagaduhan sadaya hak, hak ngabogaan tur hak kapentingan kana software ieu. Kusabab software gadung bahan atawa kode pihak katilu, pihak katilu ieu kudu meunangkeun mangpaat tina sarat ieu.

Lisensi ieu ditur ku hukum Jepang. Lamun bisa, ieu oge nerap kana hak hukum palanggan.

Dina kajadian yen software bareng atawa dibikeun nu aya kakaitna alat Anjeun eta teh dibere sarat jeung katangtuan tambahan, katangtuan eta oge ngatur hak boga jeung pamakean software Anjeun.
